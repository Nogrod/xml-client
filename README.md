# nogrod/xml-client

based on idea of [goetas-webservices/soap-client](https://github.com/goetas-webservices/soap-client) 

# Installation

## Download the Bundle

Require the bundle in your `composer.json` file:

``` bash
$ composer require nogrod/xml-client
```
